# Locos por la Pizza — Landing Template

Proyecto demo: Landing / e-commerce minimal para "Locos por la Pizza".

Edición rápida de productos: modifica `src/data/menuData.js` (const `MENU_DATA`).

Instalación y ejecución (en CMD o PowerShell con permisos adecuados):

```bash
npm install
npm run dev
```

Si PowerShell bloquea `npx` o `npm` por ExecutionPolicy, usa CMD o ejecuta PowerShell como administrador:

```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
```

El `MENU_DATA` controla todo: agrega, edita, elimina productos y la interfaz se actualizará.
